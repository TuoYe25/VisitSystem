# VisitSystem - 访客管理系统

## 项目简介
VisitSystem 是一个现代化的访客管理系统，用于管理访客登记、预约、签到等业务流程。

## 功能特性
- 访客在线预约
- 现场登记管理
- 签到/签出管理
- 访客信息统计
- 报表生成
